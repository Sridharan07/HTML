# HTML Coding Exercises

Welcome to the HTML Coding Exercises repository! 

This repository contains a collection of beginner-friendly HTML exercises designed to help learners understand and practice the fundamentals of HTML.

## Exercises Included

### 2.1 Heading Element
Learn how to use HTML heading tags:

- `<h1>`
- `<h2>`
- `<h3>`
- `<h4>`
- `<h5>`
- `<h6>`

### 2.2 Paragraph Element
Practice creating and formatting paragraphs using:

- `<p>`

### 2.3 Void Elements
Understand self-closing (void) HTML elements such as:

- `<br>`
- `<hr>`

### 2.4 Movie Ranking Project
Build a simple movie ranking webpage using:

- Headings
- Paragraphs
- Lists
- Basic HTML structure

## Learning Objectives

By completing these exercises, I will learn:

- HTML document structure
- Heading and paragraph elements
- Void elements
- Organizing content on a webpage
- Creating simple HTML projects

## Technologies Used

- HTML5
- Visual Studio Code
