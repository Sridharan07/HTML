# 3.2 Nesting and Indentation

## Objective

Learn how to create nested lists in HTML and properly indent elements to represent hierarchical relationships.

## Description

This exercise demonstrates the use of nested ordered and unordered lists to create multi-level list structures. Proper indentation improves code readability and helps organize related content within parent and child list items.

## Key Learning

* Create nested lists by placing a list inside a list item (`<li>`).
* Combine ordered (`<ol>`) and unordered (`<ul>`) lists within the same structure.
* Use indentation to improve code readability and maintainability.
* Represent hierarchical relationships using parent and child list items.
* Understand how browsers render multi-level lists.

## Output

The webpage displays a hierarchical list structure containing multiple levels of ordered and unordered lists, demonstrating proper nesting and indentation.
