# 3.4 Image Elements

## Objective

Learn how to display images on a webpage using the HTML image (`<img>`) element.

## Description

This exercise demonstrates how to add images to a webpage using the `<img>` tag. Images are an essential part of web design and help make content more engaging and visually appealing. The exercise showcases a simple webpage containing a heading and an image.

## Key Learning

* Use the `<img>` tag to display images on a webpage.
* The `src` attribute specifies the image source.
* The `alt` attribute provides alternative text for accessibility.
* Images can be displayed from local files or external URLs.
* Proper image usage improves user experience and accessibility.

## Output

The webpage displays a heading along with an image, demonstrating how images can be embedded and rendered within an HTML document.
