# 4.1 File Paths

## Objective

Learn how to reference files and folders correctly using relative file paths in HTML.

## Description

This exercise demonstrates how to link and display files located in different folders within a project directory. By using relative file paths, images can be accessed regardless of their location in the folder structure. The webpage displays various animal images stored across multiple directories.

## Key Learning

* Understand relative file paths in HTML.
* Reference files located in parent, child, and sibling folders.
* Use `../` to navigate to a parent directory.
* Organize project assets using a structured folder hierarchy.
* Display images from different locations using the `<img>` element.

## Output

The webpage displays a collection of animal images loaded from different folders, demonstrating the correct use of relative file paths.
