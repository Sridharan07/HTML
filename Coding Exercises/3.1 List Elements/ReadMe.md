# 3.1 List Elements

## Objective

Learn how to use HTML list elements to organize and display information in a structured format.

## Description

This exercise demonstrates the use of unordered lists (`<ul>`) and ordered lists (`<ol>`) in HTML. A cinnamon roll recipe is used as an example to show how ingredients can be presented as bullet points and instructions can be displayed as numbered steps.

## Key Learning

* Use `<ul>` to create unordered (bulleted) lists.
* Use `<ol>` to create ordered (numbered) lists.
* Use `<li>` to define individual list items.
* Lists help organize content and improve readability.
* Ordered lists are ideal for step-by-step instructions, while unordered lists are useful for grouped items.

## Output

The webpage displays a cinnamon roll recipe with ingredients organized in unordered lists and preparation steps displayed in an ordered list.
