# 3.3 Anchor Elements

## Objective

Learn how to create hyperlinks in HTML using anchor (`<a>`) elements to navigate users to external websites and resources.

## Description

This exercise demonstrates the use of HTML anchor elements to create clickable links. The webpage displays a list of favorite websites, where each website name acts as a hyperlink that redirects users to the corresponding external webpage.

## Key Learning

* Use the `<a>` tag to create hyperlinks.
* The `href` attribute specifies the destination URL.
* Anchor elements can link to external websites, internal pages, and other resources.
* Hyperlinks improve website navigation and user experience.
* Ordered lists can be combined with anchor elements to organize links effectively.

## Output

The webpage displays a ranked list of favorite websites, with each website name functioning as a clickable link that opens the corresponding website.
