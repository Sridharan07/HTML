# 2.2 Paragraph Element

## Objective

Learn how to use the HTML paragraph (`<p>`) element to display blocks of text on a webpage.

## Description

This exercise demonstrates the use of the `<p>` tag to organize and display textual content in a readable format. Paragraph elements help structure content by separating text into distinct sections.

## Key Learning

- The `<p>` tag defines a paragraph of text.
- Browsers automatically add spacing before and after paragraphs.
- Paragraphs improve readability and content organization.
- Multiple paragraphs can be used to separate different ideas or topics.

## Output

The webpage displays text content in separate paragraphs with appropriate spacing between them.
