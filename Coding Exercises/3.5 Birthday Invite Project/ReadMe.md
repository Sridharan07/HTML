# 3.5 Birthday Invite Project

## Objective

Apply HTML fundamentals by creating a simple birthday invitation webpage using headings, images, lists, and hyperlinks.

## Description

This project demonstrates how multiple HTML elements can be combined to create a complete webpage. The birthday invitation includes event details, an image, a list of items to bring, and a link to the event location.

## Key Learning

* Use heading elements to organize page content.
* Display images using the `<img>` element.
* Create unordered lists using `<ul>` and `<li>`.
* Add hyperlinks using the `<a>` element.
* Combine multiple HTML elements to build a complete webpage.

## Output

The webpage displays a birthday invitation containing the event date, a birthday-themed image, a list of items guests should bring, and a clickable Google Maps link to the event location.
